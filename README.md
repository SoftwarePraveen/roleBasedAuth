# roleBasedAuth
A Node.js project for implementing **Role-Based Authentication**.

## Navigate into the Project Directory
cd roleBasedAuth

## Clone the Repository
git clone https://github.com/SoftwarePraveen/roleBasedAuth.git

## npm depenedencies
npm install

## Create .env sample
.env.example .env

## Run Command
npm start
